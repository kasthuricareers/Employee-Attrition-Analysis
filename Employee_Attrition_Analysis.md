# Employee Attrition Analysis using Python, SQL & Power BI

**Project Overview**

This project focuses on analysing employee attrition trends to identify the key factors influencing workforce turnover. 
Using Python for data cleaning and exploration, SQL for analytical querying, and Power BI for interactive visualization, 
the project provides valuable HR insights to support employee retention and workforce planning.
________________________________________
**Objectives**

•	Analyse employee attrition patterns and trends. 

•	Identify factors affecting employee turnover. 

•	Examine the impact of salary, job role, experience, department, and performance on attrition. 

•	Build an interactive HR analytics dashboard for data-driven decision-making. 
________________________________________
**Tools & Technologies**

•	Python – Data Cleaning & Exploratory Data Analysis (EDA) 

•	MySQL – Employee Attrition Analysis 

•	Power BI – Dashboard Development & Data Visualization

**Project Workflow**

**1. Data Understanding**

•	Explored employee demographics and workforce attributes. 

•	Identified key factors related to attrition. 

•	Analysed employee distribution across departments, job roles, and salary levels. 

**2. Data Cleaning using Python**

•	Handled missing values 

•	Removed duplicate records 

•	Corrected data types 

•	Performed exploratory data analysis (EDA) 

•	Created insights on attrition trends and employee characteristics 

**3. Data Analysis using SQL**

•	Employee distribution analysis 

•	Department-wise attrition analysis 

•	Gender-based attrition comparison 

•	Salary slab and compensation analysis 

•	Job role and experience-based attrition trends 

•	Performance rating and overtime impact analysis

**Key Insights**

•	Research & Development recorded the highest employee attrition. 

•	Employees in lower salary slabs showed higher attrition rates. 

•	Attrition was more common during the early years of employment. 

•	Sales-related roles experienced comparatively higher turnover.

•	Workforce demographics and performance ratings provided additional insights into retention challenges.
